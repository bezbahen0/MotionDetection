## Requirements
To compile this code you need to install opencv
```
sudo apt-get install libopencv-dev 
```

## Installation
```
git clone https://github.com/bezbahen0/MotionDetection
cd MotionDetection
mkdir build 
cd build
cmake ..
make 
./MotionDetection
```

## Execution
To run this application execute this
```
./MotionDetection
```
